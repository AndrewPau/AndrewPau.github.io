Projects

Created a persistent Google Chrome extension that tracks user activity on various websites, utilizing a self-written REST API in Node.js and Express to communicate with a NoSQL database (MongoDB). Integrated user authentication for protected endpoints with Passport.js and hashed passwords with bcrypt.js.

Constructed a matrix computation program capable of all fundamental matrix operations, which includes features include computing the change of basis matrix, QR factorization, eigenvalues and eigenvectors, basis of the column and null space, and orthonormalizing a set using Gram-Schmidt.

Created a simplified version-control system similar to Git using SHA-1 hash codes, capable of add, commit, checkout, delete, branch, and merge commands. Serialized the runtime objects to maintain the state of the program across commands.
